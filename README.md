# 💰 MoneyMentor Pro

MoneyMentor Pro is a smart expense and savings tracker built using Python and Streamlit.

## 🚀 Features

- Income Tracking
- Expense Tracking
- Transaction History
- Dashboard
- Savings Goal
- Budget Planner
- Budget Alerts
- Smart Financial Tips
- Achievement Badges
- CSV Export
- Expense Pie Chart

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
